This is simple repository of scripts that will install programs on ubuntu based distros. There is no abstract logic. Each stript contains its own logic. Such strategy is used for simplicity and reliability of usage. Do not think about how other users would use this script!

All scripts are located in src folder. All installation files are located in resources folder.

You can run all of the scriptas by run-all-scripts.sh script.
